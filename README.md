# AIPen - Typecho AI 智能写作助手

> 作者：优优
> 官网：https://blog.uuhb.cn

## 插件简介

AIPen 是一款为 Typecho 博客系统开发的 AI 写作助手插件。通过集成各类 AI 服务，帮助用户快速生成高质量的文章内容。

## 主要特性

- ✨ **灵活配置**：支持任何兼容 OpenAI 格式的 API 接口
- 🎨 **多种风格**：预设多种写作风格，满足不同场景需求
- 🚀 **简单易用**：在文章编辑页面直接使用，无需切换窗口
- 📝 **智能生成**：输入简要大纲，AI 自动生成完整文章
- 🔄 **一键插入**：生成内容可一键插入编辑器

## 安装方法

1. 下载插件文件，上传到 `/usr/plugins/AIPen/` 目录
2. 登录 Typecho 后台，进入「插件」管理页面
3. 找到「AIPen」插件，点击「启用」

## 配置说明

### API 配置

- **API 请求地址**：你的 AI 服务接口地址
  - OpenAI: `https://api.openai.com/v1/chat/completions`
  - 通义千问: `https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation`
  - 其他兼容接口的自定义地址

- **API 密钥**：你的 API Key

- **模型名称**：要使用的 AI 模型
  - OpenAI: `gpt-3.5-turbo`, `gpt-4`, `gpt-4-turbo` 等
  - 通义千问: `qwen-turbo`, `qwen-plus`, `qwen-max` 等

- **最大 Token 数**：生成内容的最大长度（建议 2000）

- **温度参数**：控制生成内容的随机性（0-2，默认 0.7）

### 写作风格配置

在「写作风格预设」中配置风格，每行一个，格式：`风格名称:提示词`

示例：
```
正式:以正式、专业的语调撰写文章，适合商务场景
轻松:以轻松、幽默的语调撰写，适合博客分享
学术:以学术、严谨的语调撰写，适合研究论文
营销:以营销、推广的语调撰写，突出产品优势
```

## 使用方法

1. 进入「撰写文章」页面
2. 滚动到页面底部的「AIPen」面板
3. 选择写作风格
4. 输入文章主题或大纲
5. 点击「生成文章」按钮
6. 等待 AI 生成完成
7. 查看生成结果，满意后点击「插入编辑器」

## 常见问题

### Q: 提示 API 请求失败？

A: 请检查：
1. API 地址是否正确
2. API Key 是否有效
3. 网络连接是否正常
4. 模型名称是否正确

### Q: 生成内容质量不理想？

A: 可以尝试：
1. 调整温度参数（降低更严谨，升高更创意）
2. 提供更详细的主题和大纲
3. 自定义更适合你的风格提示词

### Q: 支持哪些 AI 服务？

A: 理论上支持所有兼容 OpenAI API 格式的服务，包括：
- OpenAI 官方 API
- 通义千问
- 文心一言
- DeepSeek
- 以及其他第三方代理服务
